# SprintList
A minimalist Kanban board for project task completion.

## Contents
- [About](#about)
- [Features](#features)
- [Tech Stack](#tech-stack)

## About
This is a lightweight, minimalist-style Kanban board for both individuals and small teams who want a way to track task progress without complexity. The focus is on clarity and speed, making it easy to capture progress at a glance, and avoid any unnecessary distractions.

## Features 
- Clean three-column workflow ("Todo", "In Progress", and "Done") with minimal visual noise.
- Quick task creation, editing, and drag-and-drop movement between columns.
- Lightweight persistence so boards load quickly and are easy to reset.

## Tech Stack
- Python 3.12
- Django 5.2
- SQLite
